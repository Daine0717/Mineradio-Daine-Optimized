# Mineradio 优化版 2.1.1 发布说明

## 项目说明

本项目由 Daine 基于原创项目 [XxHuberrr/Mineradio](https://github.com/XxHuberrr/Mineradio) 与二创项目 [zzstar101/Mineradio-Tauri](https://github.com/zzstar101/Mineradio-Tauri) 继续优化，用于个人学习、Electron 桌面应用开发和功能扩展。

本修改版继续遵循 GPL-3.0 协议，保留来源项目版权、协议和作者信息，不代表原创作者或二创作者的官方版本。

## 当前版本内容

- 网易云音乐、QQ 音乐、酷狗音乐三平台登录和账号状态展示。
- 酷狗账号信息、VIP 状态、云端歌单、歌词与播放接口接入。
- Home 页面六个中文快捷入口、“我喜欢”多平台筛选和热门搜索入口。
- `900 × 150` 极简液态玻璃歌词窗口，支持基础播放控制和一键退出。
- 极简专用低密度星河视觉，进入前保存设置，退出后完整恢复。
- 极简切换动画、屏幕边缘约束和缩放性能优化。
- 修复极简按钮闪现、退出失效、设置污染、旧尺寸闪帧和窗口定位闪烁。

完整记录见 `CHANGELOG.md`。

## 安装包

Windows 安装包名称：

`Mineradio优化版-2.1.1-Setup.exe`

重新构建：

```powershell
npm install
npm run build:win
```

## 功能边界

第三方音乐平台功能仅用于用户自有账号的登录、歌单同步、歌词同步和播放体验。本项目不提供破解会员、绕过版权限制、破解付费内容或非法下载音乐等功能。

网易云音乐、QQ 音乐、酷狗音乐等名称和商标归其权利人所有，本项目与相关平台官方不存在从属、合作或授权关系。
