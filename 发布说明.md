# Mineradio 修改版发布说明

## 项目说明

本项目是基于开源音乐播放器项目 `XxHuberrr/Mineradio` 进行学习和功能扩展后的修改版本。

原项目地址：

https://github.com/XxHuberrr/Mineradio

## 本次修改内容

相比原项目，本版本主要进行了以下修改：

1. 酷狗音乐云端同步接入
   - 在登录弹窗中新增酷狗音乐扫码 / 网页登录入口。
   - 在 Electron 桌面端新增酷狗音乐官方网页登录窗口。
   - 新增酷狗音乐 cookie 本地保存与清除功能。
   - 支持同步酷狗账号信息、VIP 状态、云端歌单、歌单歌曲顺序、歌词和播放信息。
   - 新增酷狗音乐登录状态接口：
     - `/api/kugou/login/status`
     - `/api/kugou/login/cookie`
     - `/api/kugou/login/qr/key`
     - `/api/kugou/login/qr/check`
     - `/api/kugou/logout`
     - `/api/kugou/user/playlists`
     - `/api/kugou/playlist/tracks`
     - `/api/kugou/song/url`
     - `/api/kugou/lyric`

2. 优化三平台登录界面
   - 网易云、QQ 音乐、酷狗音乐登录入口统一。
   - 已登录平台打开登录弹窗时显示账号卡片、UID 和会员状态。
   - 已登录平台可直接在登录弹窗中退出登录。
   - 优化 QQ 音乐扫码入口显示，避免看起来像未加载。

3. 扩展账号状态显示
   - 右上角账号状态支持显示酷狗音乐登录状态。
   - 支持酷狗音乐网页登录会话保存。
   - 支持酷狗音乐手动导入 cookie 会话。
   - 支持退出酷狗音乐网页登录会话。

4. 优化播放器界面
   - 替换首页“此处施工”占位为播放器启动台。
   - 优化左侧歌单面板和酷狗歌单展开详情。
   - 降低底部播放控制栏眩光，减少视觉干扰。

5. 新增修改说明文档
   - 新增 `MODIFICATIONS.md`，用于说明本修改版的主要改动和版权合规信息。

## 功能边界说明

酷狗音乐相关功能用于用户自有账号的网页登录、歌单同步、歌词同步和播放体验测试。

本项目未提供绕过会员、绕过版权、破解付费内容、非法下载音乐等功能。

如果后续继续接入搜索、歌单、播放等酷狗音乐接口，应遵守相关平台服务规则、版权要求和法律法规。

## 安装包说明

Windows 安装包已通过 Electron Builder 生成，文件位于：

`dist/Mineradio-1.1.2-Setup.exe`

如需重新打包，可执行：

```bash
npm install
npm run build:win
```

如果国内网络下载 Electron 依赖较慢，可临时使用镜像：

```powershell
$env:ELECTRON_MIRROR='https://npmmirror.com/mirrors/electron/'
$env:ELECTRON_BUILDER_BINARIES_MIRROR='https://npmmirror.com/mirrors/electron-builder-binaries/'
npm run build:win
```
