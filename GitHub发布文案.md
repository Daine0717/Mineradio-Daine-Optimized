# GitHub 仓库简介

Mineradio 优化版是基于 `XxHuberrr/Mineradio` 开发的 Windows 沉浸式音乐播放器个人修改版，支持网易云、QQ、酷狗三平台音乐体验，并加入极简歌词窗口、Home 页面优化和酷狗云端歌单同步。

原项目：https://github.com/XxHuberrr/Mineradio

本项目继续遵循 GPL-3.0 协议，不代表原作者官方版本。

## GitHub Release 标题

Mineradio 优化版 v2.1.1｜极简歌词窗口与 Home 页面升级

## GitHub Release 正文

Mineradio 优化版 `2.1.1` 基于开源项目 `XxHuberrr/Mineradio` 修改，并继续遵循 GPL-3.0 协议发布。

### 本次更新

- 新增极简模式：将播放器平滑切换为 `900 × 150` 液态玻璃歌词条。
- 极简窗口提供上一首、播放/暂停、下一首和一键退出控制。
- 新增极简专用低密度星河视觉，退出后自动恢复原视觉设置。
- 加入窗口边缘约束、切换动画和缩放性能优化，减少掉帧、拖影与闪烁。
- Home 页面新增“我喜欢”“今日推荐”“上次听到”“热门歌单”“我的播客”“我的歌单”入口。
- “我喜欢”支持网易云、QQ、酷狗平台筛选和对应红心歌单加载。
- 优化搜索热门入口、歌词布局、控制器层级和视觉控制台。
- 修复极简模式退出按钮失效、首次闪现、设置污染和窗口定位闪烁等问题。

### 下载

Windows 安装包：

`Mineradio优化版-2.1.1-Setup.exe`
