# GitHub 仓库简介

这是基于开源项目 `XxHuberrr/Mineradio` 的个人学习修改版。

原项目地址：https://github.com/XxHuberrr/Mineradio

本项目继续遵循 GPL-3.0 开源协议，仅用于学习、研究和功能扩展练习，不代表原作者官方版本。

## 修改内容

- 新增酷狗音乐登录同步基础框架。
- 登录弹窗新增“酷狗音乐”入口。
- Electron 桌面端新增酷狗官方网页登录窗口。
- 服务端新增酷狗登录状态、cookie 保存和退出接口。
- 左侧播放列表和底部播放队列新增透明玻璃感滚动条。
- 右上角账号状态支持显示酷狗音乐登录状态。
- 新增 `MODIFICATIONS.md` 和 `发布说明.md`，说明修改内容和版权合规信息。

## 说明

酷狗音乐功能目前为登录同步基础框架，主要用于保存网页登录会话和展示登录状态。

本项目不提供破解会员、绕过版权限制、非法下载音乐等功能。

# GitHub Release 发布内容

本版本基于开源项目 `XxHuberrr/Mineradio` 修改，继续遵循 GPL-3.0 协议发布。

## 更新内容

1. 新增酷狗音乐登录同步基础框架
   - 新增酷狗音乐登录入口。
   - 新增酷狗官方网页登录窗口。
   - 支持保存酷狗网页登录 cookie。
   - 支持查询和退出酷狗登录状态。

2. 优化播放列表视觉效果
   - 左侧播放列表新增透明玻璃感滚动条。
   - 底部小播放队列同步使用透明滚动条。

3. 扩展账号显示逻辑
   - 右上角账号状态支持酷狗音乐。
   - 支持酷狗音乐手动导入 cookie。
