# GitHub 仓库简介

这是基于开源项目 `XxHuberrr/Mineradio` 的个人学习修改版，重点加入酷狗音乐登录、云端歌单同步、用户信息同步和播放器界面优化。

原项目地址：https://github.com/XxHuberrr/Mineradio

本项目继续遵循 GPL-3.0 开源协议，仅用于学习、研究和功能扩展练习，不代表原作者官方版本。

## 修改内容

- 新增酷狗音乐扫码 / 网页登录入口。
- 支持酷狗账号信息、VIP 状态和云端歌单同步。
- 支持酷狗歌单歌曲顺序、歌词和播放接口接入。
- 网易云、QQ 音乐、酷狗音乐三平台登录入口统一。
- 已登录平台在登录弹窗内显示账号卡片，并可直接退出登录。
- 优化左侧歌单面板、首页空状态、QQ 登录入口和底部播放控制栏视觉效果。
- 新增 `MODIFICATIONS.md` 和 `发布说明.md`，说明修改内容和版权合规信息。

## 说明

酷狗音乐功能用于用户自有账号的登录状态、云端歌单和播放体验同步。

本项目不提供破解会员、绕过版权限制、非法下载音乐等功能。

第三方音乐平台名称和商标归其权利人所有。本项目与网易云音乐、QQ 音乐、酷狗音乐官方无任何从属、合作或授权关系。

# GitHub Release 发布标题

Mineradio 修改版 v1.1.2 - 酷狗云端歌单同步与三平台登录优化

# GitHub Release 发布内容

本版本基于开源项目 `XxHuberrr/Mineradio` 修改，继续遵循 GPL-3.0 协议发布。

## 更新内容

1. 酷狗音乐同步接入
   - 新增酷狗音乐扫码 / 网页登录入口。
   - 支持保存酷狗网页登录会话。
   - 支持同步酷狗账号信息、VIP 状态和云端歌单。
   - 支持酷狗歌单歌曲、歌曲顺序、歌词和播放接口。

2. 三平台登录体验优化
   - 网易云、QQ 音乐、酷狗音乐登录入口统一。
   - 已登录账号在登录界面显示账号卡片。
   - 已登录平台可直接退出登录，不再显示其它登录入口。

3. 界面视觉优化
   - 优化 QQ 音乐登录入口显示。
   - 替换首页“施工”占位为播放器启动台。
   - 优化左侧歌单、酷狗歌单展开详情和底部播放控制栏观感。
